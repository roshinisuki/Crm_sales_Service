# Worklog — 7 July 2026

One major design-system workstream completed today: **CRM Theme Normalization**. All color theme variants (blue, green, purple) in dark mode have been refactored to inherit the exact same polished visual logic, surfaces, and contrast system as the **Orange Dark Theme** (master reference), changing only their accent hue.

## 1. CRM Theme Normalization

**Goal:** Treat the orange dark theme as the canonical visual reference, keeping dark mode surfaces consistent across all themes so that they feel equally premium, readable, and visually balanced.

### Token Refactoring Architecture

We split the theme variables in `styles/themes.css` and `app/globals.css` into two groups:

1. **Shared Dark Foundation Tokens**: Defined in a single `:root[data-mode="dark"]` block (and mapped to legacy selectors `ember-dark`, `ocean-dark`, `forest-dark`, and `obsidian-dark` for complete backward compatibility).
2. **Accent Theme Tokens**: Overridden per theme block to control only the accent hue, active hover/pressed state, focus colors, and soft background tones.

#### Unified Dark Foundation Tokens
These remain constant across all dark themes:
- App Background (`--bg-app` / `--bg`): `#000000`
- Card Surface (`--bg-card` / `--surface`): `#0A0A0A`
- Alternative Card Surface (`--bg-card-alt` / `--surface-1`): `#181818`
- Surface Level 2 (`--surface-2`): `#1E1E1E`
- Strong Border (`--border-strong` / `--border`): `#2A2A2A`
- Subtle Border (`--border-subtle`): `#222222`
- Text Colors: Primary (`#F9FAFB`), Secondary (`#9CA3AF`), Muted (`#8B8B8B`), Disabled/Tertiary (`#6B7280`)
- Table Header: background (`#1A1A1A`), row alternate (`#161616`), hover row (`#181818`)
- Topbar: background (`#1A1A1A`), border (`#2A2A2A`)
- KPI Card states and status colors

#### Accent-Specific Color tuning
- **Orange Theme (`orange` / `ember`)**: Left intact as the master reference (primary: `#FF6901`, hover: `#FF4601`, readable text: `#FFB37F`, offset: `#2A1A10`).
- **Blue Theme (`blue` / `ocean`)**: Restructured from muddy blue surfaces. Aligned accents (primary: `#2090FF`, hover: `#0078F2`, readable text: `#94CAFF`, offset: `#0F1D33`).
- **Green Theme (`green` / `forest`)**: Fixed muddy green surfaces and the unreadable pitch-black hover state (`#234302`). Aligned accents (primary: `#7F9E15`, hover: `#6C8712`, readable text: `#C3E250`, offset: `#1A2004`).
- **Purple Theme (`purple` / `obsidian`)**: Replaced low-contrast dark purple (`#7D289F`) with vibrant purple. Aligned accents (primary: `#BD5CE9`, hover: `#A53FD0`, readable text: `#E4B5F6`, offset: `#290F33`, contrast text on primary buttons: `#1A1A1A`).
